# roblox-cr

roblox-cr is a Crystal-to-Luau compiler for Roblox. It takes a string of Crystal code and converts it into Lua.

## Installation

1. Clone the repo
2. Run `make install`
3. Check if everything is working by running `rbxcr -h`
